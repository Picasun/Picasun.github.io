答辩问题

技术相关问题

  • 前端+后端+数据库（成熟、全面、丰富）纯软件系统

    • 前端：Vue （html css js）
    • 后端：两种集成方式（分别是基于他们 和基于我们）
      • 基于原始系统的SDK，我们进行我们系统的进一步开发（相对复杂）
        • 我们参照了用的语言还是 Java（跨平台、库、框架） 以及Python
        • 我们的系统可以很好的进行集成开发 同样的平台只需要重构一下函数即可
      • 调用我们软件的接口，直接在页面进行呈现（相对简单）
        • 需要什么类型的数据 我们给出什么类型的数据即可
    • 数据库：TD Engine 自己构建
      • TDengine 是一个开源的时序数据库
      • 专为物联网、工业互联网和大数据应用场景设计。
      • 高性能增删改查，同时支持时间序列数据的高效存储和压缩
      • 并提供快速的数据分析和可视化能力。
    • 部署：阿里云

  • 入参：地形 降雨 排水（以珊格数据的方式呈现、空间分析、制图、模型构建）

    • 地形：使用地形测量仪器（如激光雷达）或遥感影像获取地表高程数据（测绘公司购买）
      • 如数字高程模型
      • 精确
    • 降雨：气象局or商业公司 （通过协议 购买）
      • 一般不会像地形那么精确 以范围的形式呈现
      • 比如雨带的内环中环外环 深度表示降雨量范围
    • 排水：水利局 or 排水能力值（城市排水系统评价标准）
      • 将排水区域划分为离散的栅格单元，每个单元表示一个水文单元或排水区域。
    • 也可以有比如道路交通，排水分区等输入
    • 其他参数：饱和导水率、蒸发系数、最大含水量亏损、曼宁系数

  • 参数处理：珊格数据的赋值、映射、建模

    • 不同的数据值映射到珊格（Raster）数据的各个像元（Pixel）或栅格单元（Cell）上
    • 珊格内部运算、之间运算
    • 珊格化方法：插值法

  • 输出

    • 可视化的图像
    • 进而进行淹没分析（上海80% 宁波70%地形复杂）

  • 预测

    • 多久我们能获取到降雨数据以及降雨数据的精确性
    • 30分钟 的处理可以预测宁波后24h的数据
    • 降水强挂钩 开始降水 启动模拟
    • 不存在中途模拟问题

  • 模型耦合

    • 二维地表淹没模型耦合下面四个
      • 城市水文模型
      • 一维河流水动力模型
      • 一维官网水动力模型
      • 二维风暴潮模型

  • 人工智能+的体现

    • 参数一开始不准确 通过模拟很多长 经验数据 找到更准确的排水参数值
    • 比如地形 比如排水

产业化相关问题

关于产业化进展：
Q1：已筹资金来源是什么？
A: 感谢学校和社会给予我们初创团队的支持啊，资金来源我们这个除了我们自筹的100万元以外，
那么我们分别得到了学校的创智汇基金的支持10万，然后与此同时也得到了上海市大学生科创基金50万元的支持，这两个是免息贷款。这样就是帮助我们在创业初期就获得了总共160万元的资金支持，目前这些资金都已到账。

Q2：上游企业提供什么样的服务？
A：上游企业提供气象类、地形，以及当地实时监控的数据；

Q3：首先要触达的目标客户是哪个群体？
A：我们最先触达的目标客户是相关的政府管理部门，目前，华东师范大学已经成立了灾害风险研究院，通过华东师范大学目前已有的联合实验室，横向项目（这里需要具体指出是什么实验室和什么横向的项目）来达成进一步的合作。

Q4：对于已有的订单，金额是否已经到账，是和上海预知然公司签订的协议吗？
A：这个问题待定，需要等公司公章下来以后再解决。

Q5：为什么意向融资企业是平安财险公司和应急管理部？
A：我们的系统平台用于对特定地区的洪涝的预报预警，对于财险公司和应急管理部而言，可以使用我们的系统精准的看到哪些地区易发生洪涝，从而为他们在投保时提供决策辅助。

Q6：总融资额一共400万，确定这个金额的标准是什么？
A：通过我现在已有签订的协议啊，24、25年预计的营收，以及我个人固定资产的持续投入，加上我现在自己的知识产权体系通过第三方前海创投孵化器做的做科学估价，整体项目估值达到四千万左右啊，因此我愿意出放10%的股权啊，去融资400万。

Q7：对接问题（学科基础接触社会 联合实验室进一步接触 产学研）（华东师范大学风险管理研究院）
A：我们的创始人以及联合创始人深度参与了项目的重要过程，依托华东师范大学A级软件工程以及地理科学学科优势以及对我们的培养让我们拥有了更好的社会认知，给予了我们接触社会的机会。因此我们才可以获得这方面的锻炼。
同时华师大和无线传感器网络实验室（上海市计算技术研究所）等成立了联合实验室，让我们也参与了很多实践工作，接触了很多关于地理信息系统，洪涝的横向课题，让我们进一步的能够接触社会，产生了产学研的想法并落地，有了实践的基础让我们更好的知道了自己在干什么并树立了信心与目标。

Q8：所有权问题
A：部分项目是过去和华东师范大学签订、其他是直接和我们公司进行签订
专利，国际知识产权局，购买心仪专利，也可以发布专利需求，寻求高校专利转让许可
专利权利人并明确许可使用费支付方式和标准、许可范围和时间，我们进行按需购买
专利 软著 归属都是公司，因为你是法人，所以其实都在你名下。
